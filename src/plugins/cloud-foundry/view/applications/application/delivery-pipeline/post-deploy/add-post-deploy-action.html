StormRunner Load Test

Simulate usage on your application to understand the resources consumed during scaling events.

If you don't have an account already:

  • Create a new Storm Runner Account
  • Note Tenant ID, Project ID and credentials
  • Create a load test and note your Test ID
  • Come back and enter your details in this form

After set-up, every deploy will trigger a load test to help you measure their effect on the scaling behavior of your app.
